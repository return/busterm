
# busterm
View all the NapTAN buses directly in realtime in the terminal!

![](preview.png)

If you have Go installed:

`$ go get github.com/return/busterm`

Release binaries will be available soon.

```
Usage:
	busterm [-t] -n <code> | --naptan <code> [<interval>] 
	busterm -a | --api
	busterm -h | --help
	busterm --version
```
